# Organisation des Workflows

Ce dossier est séparé par **niveau de maturité** :

- `stable/` : workflows validés, documentés et prêts à être intégrés dans des projets BMAD.
- `incubation/` : workflows en cours de création, d'expérimentation ou de test.


## Flux recommandé

1. Créer un nouveau workflow dans `workflows/incubation/<nom-workflow>/`.
2. Tester, itérer et finaliser la documentation.
3. Promouvoir vers `workflows/stable/<nom-workflow>/` quand il est prêt.
